# 🗣️ AI Voice Typer & Translator

A Continuous Voice-to-Text and Translation tool built with Python. This application listens to your voice in Hindi, translates it to English in real-time, and automatically types it into any active text window (Notepad, WhatsApp, Word, VS Code, etc.). 

It features a modern Dark Mode GUI using Tkinter and allows you to save your entire transcription history to a `.txt` file.

## ✨ Features
- **Continuous Listening:** Speak as much as you want; the tool keeps listening until you stop it.
- **Real-Time Translation:** Accurately translates Hindi speech to English using `deep-translator`.
- **Auto-Typing:** Simulates keyboard strokes to type the translated text directly wherever your cursor is placed.
- **Dark UI & History Box:** A clean interface that shows exactly what was heard (Hindi) and what was typed (English).
- **Save Transcription:** Easily save your entire conversation history to a text file with a single click.
